Student Table Management (React)
📌 Project Overview

This project is a simple Student Management System built using React.
It allows users to add, edit, delete, and view student details in a table format.
The application also supports exporting student data to an Excel file.

🚀 Features

* Add new student details

* Edit existing student information

* Delete student records

* Display students in a dynamic table

* Download student data as Excel file

* Simple and responsive UI

Technologies Used

* React.js

* JavaScript

* HTML5

* CSS3

* XLSX library (for Excel export)

* File Saver
